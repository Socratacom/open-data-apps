RequireJS!

Shuffle is compatible with AMD loaders like RequireJS. You can take a peek at the config file used on this page. Just remember, Shuffle needs Modernizr to work! I also don't recommend using the "jquery.shuffle.modernizr.js" file with RequireJS because Modernizr doesn't have an AMD defintion block.
